始まりはただ一つの命令から――

## 世界線を支配する4つの究極的制御要素

### 1. メソッドとパス - 命令の型式！

フゥーハハハ！この分類こそ、世界の命令系統だッ！

| メソッド | 意味 |
|----------|------|
| GET | 情報を献上せよ！ |
| POST | 新しき情報を刻み込めッ！ |
| PUT | 既存の情報を我が意志で上書け！ |
| DELETE | 無用な情報を、世界から抹消せよ！ |
| HEAD | 中身までは要らん、輪郭のみを寄越せ！ |

---

### 2. フィールド（ヘッダー） - 情報管理の要塞！

古の電子メールから受け継がれし、**完全なる情報統制装置**ッ！

**リクエストヘッダー：**
- `User-Agent`: 我が端末の名はChrome……記憶せよ。
- `Accept`: あらゆるMIMEタイプを受け入れる覚悟はできている。
- `Authorization`: 我が身分証明書を添付するッ！

**レスポンスヘッダー：**
- `Content-Type`: HTMLであると宣言する。
- `Content-Length`: データサイズは32バイト。
- `Date`: 支配者が刻む"時"。

---

### 3. コンテンツ（ボディ） - 実体の顕現！

その身に情報の真髄を宿す、**真の兵器**！

```
Content-Length: 32

<html><body>hello</body></html>
```

空行の向こうに広がる情報の海……そこは、選ばれし者のみが干渉を許される聖域！

---

### 4. ステータス - 支配者の裁定！

命令を下したクライアントに対し、支配者サーバーが下す審判！

| ステータス | 意味 |
|------------|------|
| 200 OK | 命令は完遂された。 |
| 404 Not Found | 愚か者よ、その情報は存在しない…… |
| 301 Moved Permanently | 永遠の移転だ、従え。 |
| 302 Found | 一時的な逃亡、いずれ戻るだろう。 |
| 500 Internal Server Error | 支配者にすら読めぬ混沌が発生した…… |

## リダイレクトという世界線転移！

### リダイレクトとは――移転の儀式！

- `301 Moved Permanently`: 新たなる定住先。運命は定められた！
- `302 Found`: 一時避難。やがて収束するだろう。

**実際の転移の流れ：**

```
[ブラウザ] --(リクエスト)--> [旧サーバー]
[旧サーバー] <--(301 + Location)-- [ブラウザ]
[ブラウザ] --(新URLにリクエスト)--> [新サーバー]
[新サーバー] <--(200 OK + コンテンツ)-- [ブラウザ]
```

まるで世界線の再移動……その一連の干渉過程を目に焼き付けろ！

## URLの構造 - 世界の"座標"！

```
https://www.sweets-paradise.com:443/search/cake?flavor=strawberry#ranking
```

| 要素 | 意味 |
|------|------|
| scheme | https - 暗号化通信という名の盾 |
| host | www.sweets-paradise.com - 宿主サーバーの正体 |
| port | 443 - 通信の門番番号 |
| path | /search/cake - 階層構造による位置情報 |
| query | flavor=strawberry - クエリの名を借りた呪文 |
| hash | ranking - ページ内の指定ポイント（サーバーは知らぬ） |

✅ **ハッシュは、世界線の内側でのみ意味を持つ。干渉されることなき存在だ！**

## MIMEタイプとContent-Type - 情報の属性を示せ！

これは、かつて電子メール界からもたらされた**"ファイルの真名"**だ！

| MIMEタイプ | 内容 |
|------------|------|
| text/html | HTMLファイル |
| text/plain | 通常のテキスト |
| application/json | JSON構造体 |
| image/jpeg | JPEG画像 |
| application/octet-stream | 未知のバイナリ兵器 |

例：

```
Content-Type: text/html; charset=utf-8
```

UTF-8でエンコードされた、HTML文書であることを宣言する文言だ！

## curlコマンドで世界を覗け！

```bash
curl http://localhost:18888/greeting
curl -v http://localhost:18888/greeting
curl -X POST -d "hello=world" http://localhost:18888
curl -d '{"hello": "world"}' -H "Content-Type: application/json" http://localhost:18888
```

| オプション | 意味 |
|-----------|------|
| -v | 詳細な干渉ログを吐き出す |
| -X | メソッドの強制指定 |
| -d | ボディのデータ指定 |
| -H | ヘッダー追加 |
| -L | リダイレクト追従 |
